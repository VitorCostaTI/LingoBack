# Back End - Sistema Admnistrativo Lingo Lingo Kennel.

Este Projeto foi criado com [Spring Boot](https://start.spring.io/).

## Iniciando Scripts.

* Você precisará configurar um Banco de Dados PostgreSQL em 'src/main/application.properties'.

* Você poderá ultiliza-lo apenas dando play na IDE de desenvolvimento IntelliJ, Configure o build da maneira que achar melhor.

Inicia Projeto pelo modo de desenvolvimento.\
Recomendo testa-la [Postman](https://web.postman.co/home), foi o software que usei para validar os EndPoints.

Esta aplicação está em constante mudança então lembre-se de mante-la sempre atualizada e seguir as dicas aqui disponiveis.

* Obs: O front End deste projeto está disponivel outro [Repositório](https://github.com/VitorCostaTI/LingoFront), de meu perfil.
